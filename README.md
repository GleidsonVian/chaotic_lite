# Chaotic Lite

Uma versão simplificada do jogo Chaotic, desenvolvida para aprendizado e diversão.

## Como jogar localmente

1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web.

## Estrutura do Projeto

- `/src/css/`: Arquivos de estilo (CSS).
- `/src/js/`: Lógica principal do jogo.
- `/src/js/data/`: Dados das cartas e elementos do jogo.
- `/src/assets/`: Imagens e ícones.

## Divisão de Tarefas

**Para o Aprendiz (Front-end e Dados):**
- Modificar o `style.css` para deixar o jogo bonito.
- Adicionar novas cartas no arquivo `src/js/data/cards.js`.
- Criar a estrutura visual das cartas no HTML.

**Para o Instrutor (Lógica do Jogo):**
- Desenvolver o motor de combate no `main.js`.
- Gerenciar os turnos e estados do jogo...
